# Docker Inference Server Release Notes

Release notes for the docker inference server.

See https://ai.google.dev/gemini-api/docs/eap/robotics/robotics-ondevice#setup-model-server
for how to download and use the docker inference server.

## 1.2.1

- Includes bugfixes and improved error messaging.

## 1.2.0

- Allows reading names used during training directly off of the checkpoint, avoiding the need for a separate config file.

## 1.1.0

- Adds support for customization of image names and proprioception key names, read off of a separate config file.

## 1.0.0

- Initial release of inference server.
